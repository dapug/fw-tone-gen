# FW Tone Generator

### A tone generator for the TD-H3 using nicFW v2.52.20+

#### How to use:

1. Download VS Code 
2. Install NodeJS and Vue 3
3. Clone the repo
4. From your repo folder:
  - ```npm install```
  - ```npm run dev```
